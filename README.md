# Notification SignalR
NotificationSignalR is a lightweight ASP.NET Core application that provides real‑time notifications using SignalR. It exposes a SignalR hub for server↔client event delivery, CORS configured for a local frontend envirement. The Angular frontend is in a notificationFrontend  repository , allowing independent development and deployment of the UI layer.

## Features
 - 	Real-time notifications via SignalR
 - 	CORS configured for http://localhost:4200

## Technologies
- **Backend**: ASP.NET Core Minimal API
- SignalR

## Installation and Setup
Clone the repository:
``` bash
git clone https://github.com/Rajfex/NotificationSignalR.git
```
Open the solution file ```(.sln)``` in Visual Studio.

Build and run the application.
